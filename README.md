## ✨ Features

- 🎬 **Preloader** — branded "MK" animation on page load
- 🖱️ **Custom Cursor** — dot + follower cursor with hover scaling (desktop only)
- 📊 **Scroll Progress Bar** — top-of-page reading indicator
- 🌌 **Particles.js Background** — interactive canvas particle network in the hero section
- ✍️ **Typed.js** — cycling role titles (FullStack Developer, React Developer, etc.)
- 🃏 **Vanilla Tilt** — 3D tilt + glare effect on project cards
- 📜 **AOS (Animate On Scroll)** — fade/zoom entrance animations throughout
- 🔗 **Active Nav Highlighting** — IntersectionObserver tracks current section
- ✅ **Contact Form Validation** — real-time blur & input validation with Bootstrap feedback
- 🔔 **Toast Notification** — success message on form submission
- ⬆️ **Back to Top** button
- 📱 **Fully Responsive** — mobile-first layout using Bootstrap 5 grid

## 🛠️ Tech Stack

**Frontend**
- HTML5, CSS3, JavaScript (ES6+)
- Bootstrap 5.3

**Libraries**
- [AOS](https://michalsnik.github.io/aos/) — scroll animations
- [Typed.js](https://mattboldt.com/demos/typed-js/) — typewriter effect
- [Particles.js](https://vincentgarreau.com/particles.js/) — particle background
- [Vanilla Tilt](https://micku7zu.github.io/vanilla-tilt.js/) — 3D card tilt
- [Font Awesome 6](https://fontawesome.com/) — icons
- [Google Fonts — Inter](https://fonts.google.com/specimen/Inter)

## 🚀 Getting Started

No build tools or dependencies to install. Just open in a browser.

```bash
# Clone the repository
git clone https://github.com/Manikantakona0/<repo-name>.git

# Open in browser
open index.html
```

Or simply drag and drop `index.html` into any browser.
